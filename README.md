## 快速安装 | Setup

1. **Clone**:
   ```bash
   git clone https://github.com/42kyx28zkf-ai/China-Combat-Decision-System
   ```
2. **Server (MCP)**:
   将以下配置加入您的 Claude/Cursor 配置文件：
   ```json
   {
     "mcpServers": {
       "ccds": {
         "command": "python3",
         "args": ["/PATH/TO/CCDS/mcp/server.py"]
       }
     }
   }
   ```

## 主权边界 | Governance

- **唯实论**：不符合物理真相和实战结果的决策直接熔断。
- **孤立性**：单次决策审计不干扰底层系统稳定性。
- **最终判断**：CCDS 提供战略参考，人类统帅拥有最终发射权。
