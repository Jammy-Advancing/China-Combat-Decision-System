## 🚩 Overview / 概览

**CCDS (China Combat Decision System)** is a world-class strategic orchestration framework. It translates the philosophical wisdom of **Top Strategists** and the pragmatic implementation of **Chief Designers** into computational protocols. 

**中国实战决策系统 (CCDS)** 是一套顶级战略编排框架。它将中国顶级战略家的辩证智慧与务实工程方法论转化为 AI 可调用的计算协议，为 Agent 提供掌控全局的“大脑”与精准落地的“骨架”。

---

## 🏛️ The Lineage / 思想溯源

CCDS 不是空中楼阁，它植根于中国近现代最成功的两套实战方法论：

1.  **Strategic Soul (战略之魂)**：源自**毛泽东战略辩证法**。核心解决“看清方向”的问题。通过《矛盾论》定位主要矛盾，通过《持久战》规划发展节奏，通过“不对称作战”实现以弱胜强。
2.  **Operational Base (实战之基)**：源自**邓小平务实工程论**。核心解决“落地拿结果”的问题。秉持“猫论”的唯实标准，通过“摸着石头过河”进行低风险试错与快速规模化。

---

## 🎯 Problem Statement / 解决什么问题

在 AI Agent 遍地走、代码生成过剩的时代，**“决策质量”**成为了核心瓶颈。CCDS 旨在解决：
- **战略迷茫**：在复杂的市场/技术环境中找不到核心发力点。
- **执行悬空**：设计了完美的蓝图，却在第一步就因为风险过大而停滞。
- **资源空耗**：由于没有定位“主要矛盾”，导致在次要细节上消耗了 80% 的精力。

## 🛠 Usage / 使用方法 (MCP)

CCDS is native to the **Model Context Protocol**. 

1. **Quick Start**:
   ```bash
   git clone https://github.com/42kyx28zkf-ai/China-Combat-Decision-System
   cd China-Combat-Decision-System
   pip install -r requirements.txt
   ```

2. **Server Configuration**:
   Add this to your `claude_desktop_config.json`:
   ```json
   {
     "mcpServers": {
       "ccds-engine": {
         "command": "python3",
         "args": ["/PATH/TO/China-Combat-Decision-System/mcp/server.py"]
       }
     }
   }
   ```
