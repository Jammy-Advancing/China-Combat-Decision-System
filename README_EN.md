## Quick Setup

1. **Clone**:
   ```bash
   git clone https://github.com/42kyx28zkf-ai/China-Combat-Decision-System
   ```
2. **Server (MCP)**:
   Add the following configuration to your Claude/Cursor profile:
   ```json
   {
     "mcpServers": {
       "ccds": {
         "command": "python3",
         "args": ["/PATH/TO/CCDS/mcp/server.py"]
       }
     }
   }
   ```

## Sovereign Governance

- **Pragmatism**: Any decision failing the test of physical truth or practical combat results is immediately terminated.
- **Isolation**: Single-decision audits shall not compromise the underlying system stability.
- **Final Judgment**: CCDS provides strategic intelligence; the Human Commander retains the exclusive right to launch.
